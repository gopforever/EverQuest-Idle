# ⚔️ EverQuest Idle

> *An idle game inspired by the original EverQuest (1999 Vanilla) — built to last forever.*

EverQuest Idle is a long-term, ever-expanding idle/incremental game that captures the spirit of classic EverQuest. Starting from the original vanilla release, the game will grow over time to include expansions, new systems, and deeper mechanics — just like the MMORPG that inspired it.

---

## 🎯 Vision

- Recreate the feel of **classic EverQuest (1999)** in idle game format
- All **14 original classes**, all **vanilla zones**, all races — faithfully represented
- Incremental progress, passive grinding, and deep progression systems
- Expansion content (Kunark, Velious, etc.) added over time — this game will **never stop growing**
